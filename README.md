### Question

**Q:** *How are graceful degradation and service workers related?*

**A**: Service workers are one way to help your program implement the concept of graceful degradation. Graceful degradation talks about how to maintain the correct functionality of the program even when some components start failing. Service workers, for instance, help keep components cached in order to guarantee interactivity even with a bad connection or no connection. It is meant to keep the program usable and accessible even in bad conditions.
