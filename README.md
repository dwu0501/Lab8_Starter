### Question

**Q:** *How are graceful degradation and service workers related?*

**A**: Service workers are one way to help your program implement the concept of graceful degradation. Graceful degradation talk about how to maintain the correct functionality of the program even when some components start failing. Service workers, for instance, help keep components in cache in order to guarantee interactiveness even with bad connection or no connection. It is to keep keep the program usable and accesible even in bad conditions.
